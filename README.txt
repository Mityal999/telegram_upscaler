ДЛЯ СОЗДАНИЯ НОВОГО ТУРНИРА ТРЕБУЕТСЯ:

1. Написать /clear (ПОСЛЕ ЭКСПОРТА, В ПРОТИВНОМ СЛУЧАЕ ВСЕ ДАННЫЕ БУДУТ УТЕРЯНЫ).
2. Выключить бота.
3. Открыть constants.txt и установить новый TOURNAMENT_END_DATE.
4. Запустить бота.

------------------------------------------------------------------------------------------

Настраиваемые значения в constants.txt:


AVAILABLE_CURRENCIES - Список всех доступных валют, которые отслеживает бот. (Пока не рассчитано на другие валюты)

CURRENCY_UPDATE_PERIOD - Частота обновления курсов валют и лимитных ордеров в секундах. (Значения меньше 5 секунд могут всё сломать)

START_BALANCE - Стартовый баланс пользователя при регистрации и на старте турнира.

ROUND_PRECISION - Количество знаков после запятой при округлении. (Влияет только на вывод. расчёты производятся с полной точностью)

TOURNAMENT_END_DATE - Дата окончания турнира.

------------------------------------------------------------------------------------------

Команды админа:
/export (Экспортирует всю информацию в новую папку с датой)
/clear (Очищает ДБ)
/add_admin (Добавить админа по ID)
/admins (Вывести список админов)
/add_balance (Докинуть $ себе)
